# react-lib-template

React Library Template


## Usage

* With Git

```bash
git clone https://github.com/Devahoy/react-lib-template my-awesome-lib
cd my-awesome-lib
./scripts/setup.sh
```

* With CURL

```bash
curl https://raw.githubusercontent.com/Devahoy/react-lib-template/master/scripts/setup.sh | bash
```
